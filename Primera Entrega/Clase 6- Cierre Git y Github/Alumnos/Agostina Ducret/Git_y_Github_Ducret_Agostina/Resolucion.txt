a. ¿Quien invento el sistema de control de version Git y por que?

Git fue inventado por Linus Torvalds. Se invento para posibilitar la ramificacion y el etiquetado como procesos de primera importancia y las operaciones que afectas a las ramas y las etiquetas tambien se almacenan en el historial de cambios.

b. ¿A quien pertenece actualmente Github y porque?

Actualmente GitHub pertenece a Microsoft, el mismo lo compro el 4/6/2018. Lo adquirio porque se adapato a una nueva estrategia, mas abierta al undo del software libre y a la integracion con otras plataformas.
