a. ¿Quién inventó el sistema de control de versión Git y por qué?
Git lo inventó Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.


b. ¿A quién pertenece actualmente Github y por qué?
El dueño actualmente es Microsoft.


c. ¿Hay otra forma que no sea la terminal para trabajar con Github?
Si, desde VSC o desde adentro de github.
