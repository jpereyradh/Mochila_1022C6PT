1) ¿Quien invento el sistema de control de version Git y porque?

	git fue inventado por linus Torvalds. el lo invento porque antes habia otro controlador de versiones (SVN) que era gratis. este controlador despues se volvio pago y al ver esto, linus decidio crear su propio controlador de versiones gratis.
	ademas se creo git pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.
