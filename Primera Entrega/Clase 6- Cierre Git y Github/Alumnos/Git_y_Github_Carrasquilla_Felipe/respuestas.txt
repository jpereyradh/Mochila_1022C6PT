a. ¿Quién inventó el sistema de control de versión Git y por qué?
El sistema de control de versión Git fue inventado por Linus Torvalds (Creador de Linux) porque BitKeeper,el sistema anterior que se usaba para controlar las versiones y que además se usó durante buen tiempo para el desarrollo del kernel de Linux, anunció que no seguiría ofreciendo una versión del software gratuita para los desarrolladores.
