A) ¿Quién inventó el sistema de control de versión Git y por qué?

El sistema de control de versiones GIT, es un pruyecto de codigo abierto que desarrollo originalmente Linus Torvalds (Creador del kernel, del sistema operativo linuz en 2005).
En 2005 la herramienta Bitkeeper (Que tambien era un sistema de control de versiones utilizada por linux) dejo de ser ofrecida de manera gratuita, esto hizo que la comunidad de desarrollo de linux se motivara a desarrollar su propia herramienta basada en los principios de Bitkeeper con el objetivo de que esta herrmienta tuviera:
- Velocidad
- Diseño sencilo
- Gran soporte para desarrollo no lineal (Es decir que existan varias ramas paralelas a la principal)
- Capaz de manejar grandes proyectos (Como el kernel de linux) eficientemente
